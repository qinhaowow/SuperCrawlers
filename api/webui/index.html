关于 SuperCrawler

SuperCrawler 是一个多平台社交媒体爬虫管理系统，支持抓取多个国内外社交媒体平台的数据。

功能特性

  • 支持多个社交媒体平台
  • 提供Web界面管理
  • 支持任务调度
  • 多种数据存储格式
  • 代理支持
  • 系统监控

支持的平台

  • 国内平台：小红书、抖音、快手、哔哩哔哩、微博、百度贴吧、知乎
  • 国际平台：Facebook、Twitter、Instagram、YouTube
